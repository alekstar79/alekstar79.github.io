# Image Search

![rewiew](review.gif)

[DEMO](https://alekstar79.github.io/image-search/)

Clone this repo:
````shell
git clone git@github.com:alekstar79/image-search.git
````
Go to project directory:
````shell
cd image-search
````
Project setup
````shell
yarn install
````
Compiles and hot-reloads for development
````shell
yarn run serve
````
Compiles and minifies for production
````shell
yarn run build
````
