# Flexbox Playground

[Demo](https://alekstar79.github.io/flexbox-playground)

## Intro

Flexbox playground with Vue(JSX).

### Flex Container
- flex-direction
- flex-wrap
- justify-content
- align-items
- align-content

### Flex Items
- flex-grow
- flex-shrink
- flex-basis
- order
- align-self
