Архитектура приложения использует модульный подход <script type="module"> В связи с чем, для запуска сценария, необходим сервер. Например, Live Server для VS Code. Npm пакет serve, позволяет быстро развернуть локальный сервер.

Mystical Tentacles

Pure Javascript application that demonstrates the functionality of canvas. When developing the application, code decomposition and an object-oriented approach were applied.

Installation and use

You can install the script by cloning this repository and running (dev): npm run serve or (prod): npm run build It is also possible to pull the image from the docker hub: docker pull alekstar79/mystical and run: docker run -d -p 80:80 -e DOCKER_ENV=true --rm --name mystical alekstar79/mystical

For ease of use, a Makefile has been created with short commands pull, run, stop.
